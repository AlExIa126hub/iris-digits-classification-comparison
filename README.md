# ML_Project4_KNN_Logistic-Regression
The aim of this project is to compare the performance of KNN and Logistic Regression on the Iris and Digits datasets. This comparison is done:
* for the original Iris dataset;
* for the Iris dataset in which the number of data points gradually decreases for all classes (-10%, -20%,
etc)
* for the Iris dataset in which the number of data points gradually decreases for one class.

Both models were optimised before comparing performances.
